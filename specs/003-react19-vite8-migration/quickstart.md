# Quickstart: React 19 + Vite 8 Migration

**Branch**: `003-react19-vite8-migration`  
**Date**: 2026-04-06

---

## Overview

This migration upgrades the frontend from React 18 + Vite 7 to React 19 + Vite 8. It is a pure dependency upgrade — no new features, no UI changes.

---

## Migration Steps

### Step 1: Run React 19 Migration Codemod

From the `frontend/` directory:

```bash
npx codemod@latest react/19/migration-recipe
```

This is a no-op if no deprecated patterns are present. Safe to run.

### Step 2: Fix Manual Breaking Change

In [frontend/src/components/ProtectedRoute.tsx](../../../frontend/src/components/ProtectedRoute.tsx), change:

```ts
// Before
export function ProtectedRoute({ children }: { children: JSX.Element }) {
```

```ts
// After
export function ProtectedRoute({ children }: { children: React.ReactNode }) {
```

`JSX.Element` from the global JSX namespace is removed in React 19. `React.ReactNode` is the idiomatic replacement for "any valid React child".

### Step 3: Update `package.json`

In `frontend/package.json`, update the following version ranges:

**dependencies:**
```json
"react": "^19.2.4",
"react-dom": "^19.2.4",
"lucide-react": "^1.7.0"
```

**devDependencies:**
```json
"@types/react": "^19.0.0",
"@types/react-dom": "^19.0.0",
"@vitejs/plugin-react": "^6.0.1",
"vite": "^8.0.3"
```

Also remove `"esbuild"` from `pnpm.onlyBuiltDependencies`:
```json
"pnpm": {
  "onlyBuiltDependencies": []
}
```

### Step 4: Reinstall Dependencies

```bash
cd frontend
pnpm install
```

This updates `pnpm-lock.yaml`. Commit both `package.json` and `pnpm-lock.yaml`.

### Step 5: Run TypeScript Types Codemod

```bash
npx types-react-codemod@latest preset-19 ./src
```

Fixes type-level breaking changes in `@types/react@19` (ref callbacks, `useRef` argument, etc.). No-op if nothing applies.

### Step 6: Verify TypeScript Compilation

```bash
pnpm build
```

Fix any remaining TypeScript errors surfaced by the new `@types/react@19` definitions.

### Step 7: Verify Development Server

```bash
pnpm dev
```

Open `http://localhost:5173` and verify all pages load without console errors.

### Step 8: Verify Production Build

```bash
pnpm build
pnpm preview
```

Confirm the build completes without errors and the preview serves correctly.

---

## Expected Outcome

- `react` and `react-dom` at `^19.2.4`
- `vite` at `^8.0.3`
- `@vitejs/plugin-react` at `^6.0.1` (Oxc-based, no Babel)
- `@types/react` and `@types/react-dom` at `^19.0.0`
- `lucide-react` at `^1.7.0`
- Dev server and production build work with no errors
- `pnpm-lock.yaml` committed with updated hashes
